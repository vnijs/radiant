Radiant – Business analytics using R and Shiny

Radiant code

Built using Shiny by Rstudio. Code available on GitHub

Package dependencies

Package dependencies: car, gridExtra, GPArotation, psych, wordcloud, AlgDesign, knitr, ggdendro, lubridate, ggplot2, pryr, shiny, magrittr, tidyr, dplyr, broom, shiny, rmarkdown, shinyAce, DT, radiant
